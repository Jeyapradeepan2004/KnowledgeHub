# KnowledgeHub

KnowledgeHub is a modern knowledge management platform built for developers to organize, search, and retain programming knowledge.

Instead of storing notes across notebooks, markdown files, browser bookmarks, and random text files, KnowledgeHub provides a centralized workspace where developers can write notes, organize them with tags, search instantly, track learning progress, and build their own personal knowledge base.

This project is being built as a full-stack backend-focused application to demonstrate real-world software engineering practices using FastAPI and PostgreSQL.

## Getting Started

### Clone the repository

```bash
git clone https://github.com/yourusername/KnowledgeHub.git
```

### Navigate into the project

```bash
cd KnowledgeHub
```

### Create a virtual environment

```bash
python -m venv venv
```

### Activate the virtual environment

Windows

```bash
venv\Scripts\activate
```

Linux / macOS

```bash
source venv/bin/activate
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
uvicorn main:app --reload
```

Open your browser:

```
http://127.0.0.1:8000
```


## Why I Built This

As a developer, I realized my notes were scattered across notebooks, text files, browser bookmarks, and documentation. Finding something I had learned weeks earlier often took longer than learning it again.

KnowledgeHub is my solution to that problem.

The goal is to build a centralized platform where developers can store, organize, search, and revisit everything they learn while also exploring modern backend development with FastAPI and PostgreSQL.
